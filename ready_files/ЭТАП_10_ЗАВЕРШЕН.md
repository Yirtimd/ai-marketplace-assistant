## Функциональность

### Навигация по разделам

Реализован sidebar с 6 разделами roadmap:

- Dashboard
- Products
- Reviews
- Content AI
- Inventory
- Settings

### Интеграция с backend API

Desktop приложение работает как API-клиент:

- health/status checks
- workflows calls
- action layer calls
- AI provider checks

### Настройки подключения

В Settings добавлена настройка `API Base URL`  
(по умолчанию: `http://localhost:8000/api/v1`).

## Архитектурное соответствие
