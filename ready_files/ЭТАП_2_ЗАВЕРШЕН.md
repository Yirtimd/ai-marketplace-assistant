## Доступные endpoints

**Products:**
- `GET /api/v1/products/` — список товаров
- `GET /api/v1/products/{id}` — товар по ID
- `GET /api/v1/products/reference/*` — справочники

**Feedbacks:**
- `GET /api/v1/feedbacks/` — список отзывов
- `POST /api/v1/feedbacks/{id}/answer` — ответить
- `GET /api/v1/feedbacks/questions` — вопросы
- `GET /api/v1/feedbacks/stats/*` — статистика

**Sales:**
- `GET /api/v1/sales/` — продажи
- `GET /api/v1/sales/orders` — заказы
- `GET /api/v1/sales/stocks` — остатки

## Использование

```python
from backend.services import get_wb_service

wb = get_wb_service()

# Товары
products = await wb.get_products(limit=10)

# Отзывы
feedbacks = await wb.get_feedbacks(is_answered=False)

# Продажи
sales = await wb.get_sales(limit=100)
```
