## Возможности AI Service

`AIService` реализует:

- `generate_text()`:
  - приоритет провайдеров: **DeepSeek -> OpenAI -> Anthropic**
  - fallback между провайдерами при ошибках
- `generate_images()`:
  - OpenAI Images API при наличии ключа
  - fallback в queued payload при отсутствии/ошибке
- `generate_video()`:
  - унифицированный payload для внешнего video provider (очередь)

Настройки провайдеров вынесены в `Settings`:

- `DEEPSEEK_API_KEY`, `DEEPSEEK_BASE_URL`, `DEEPSEEK_MODEL`
- `OPENAI_API_KEY`, `OPENAI_MODEL`, `OPENAI_IMAGE_MODEL`
- `ANTHROPIC_API_KEY`, `ANTHROPIC_MODEL`

## Интеграция в workflow

### ProductCreationWorkflow
- AI доработка описания товара через `generate_text()`
- генерация изображений через `generate_images()`
- формирование video payload через `generate_video()`

### ReviewWorkflow
- AI улучшение шаблонного ответа на отзыв через `generate_text()`
- сохранен deterministic fallback (template reply)

### SalesAnalysisWorkflow
- AI аналитический блок `ai_insights` для менеджера

### PricingWorkflow
- AI объяснение pricing-решения `ai_explanation`
